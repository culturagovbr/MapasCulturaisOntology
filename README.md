# MapasCulturaisOntology
Plugin para aplicar a ontologia da gestão cultural ao Mapas Culturais

## Ativação

Para ativar este plugin, adicione a seu config.php

```PHP

'plugins' => [

    //... other plugin you may have...
    'MapasCulturaisOntology' => [
        'namespace' => 'MapasCulturaisOntology',
    ],
],

```
